## ðŸŽ“ Certification Preparation

### Docker Certified Associate (DCA)

**Exam Info:**
- Cost: $195
- Duration: 90 minutes
- Questions: 55
- Passing: 65%

**Preparation:**
- [DCA Exam Guide](https://training.mirantis.com/certification/dca-certification-exam/)
- [DCA Practice Exams](https://www.udemy.com/course/docker-certified-associate-exam-course/)
- Study Time: 2-3 months

**Topics:**
- Orchestration (25%)
- Image Creation (20%)
- Installation and Config (15%)
- Networking (15%)
- Security (15%)
- Storage and Volumes (10%)

### Certified Kubernetes Administrator (CKA)

**Exam Info:**
- Cost: $395 (includes retake)
- Duration: 2 hours
- Format: Performance-based
- Passing: 66%

**Preparation:**
- [CKA Curriculum](https://github.com/cncf/curriculum)
- [CKA Course - Udemy](https://www.udemy.com/course/certified-kubernetes-administrator-with-practice-tests/)
- Study Time: 3-4 months

**Topics:**
- Storage (10%)
- Troubleshooting (30%)
- Workloads & Scheduling (15%)
- Cluster Architecture (25%)
- Services & Networking (20%)

### Certified Kubernetes Application Developer (CKAD)

**Exam Info:**
- Cost: $395 (includes retake)
- Duration: 2 hours
- Format: Performance-based
- Passing: 66%

**Topics:**
- Application Design (20%)
- Application Deployment (20%)
- Application Observability (18%)
- Application Environment (16%)
- Application Maintenance (20%)
- Services & Networking (13%)

### AWS Certified Solutions Architect - Associate

**Exam Info:**
- Cost: $150
- Duration: 130 minutes
- Questions: 65
- Passing: 720/1000

**Preparation:**
- [AWS Training](https://aws.amazon.com/certification/certified-solutions-architect-associate/)
- [Practice Exams - Tutorials Dojo](https://tutorialsdojo.com/)
- Study Time: 2-3 months

**Topics:**
- Design Resilient Architectures (30%)
- Design High-Performing Architectures (28%)
- Design Secure Applications (24%)
- Design Cost-Optimized Architectures (18%)

### AWS Certified DevOps Engineer - Professional

**Exam Info:**
- Cost: $300
- Duration: 180 minutes
- Questions: 75
- Passing: 750/1000

**Prerequisites:**
- Associate-level certification recommended
- 2+ years AWS experience

## ðŸ’¡ Tips for Using These Resources

1. **Don't Try to Learn Everything**
   - Focus on fundamentals first
   - Deep knowledge > broad surface knowledge

2. **Practice More Than Reading**
   - 80% hands-on, 20% theory
   - Build projects while learning

3. **Join Communities Early**
   - Ask questions
   - Help others (best way to learn)

4. **Stay Updated**
   - Follow blogs and newsletters
   - Attend webinars and conferences

5. **Set Clear Goals**
   - Pick certification path
   - Build portfolio projects
   - Update resume regularly

---

**Remember:** The best resource is the one you actually use. Pick 2-3 and stick with them!
