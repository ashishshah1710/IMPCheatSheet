## ğŸ¯ File Naming Conventions

### README Files
- Always named `README.md`
- Located in each folder
- Contains complete tutorial/guide

### Cheat Sheets
- Named `CHEATSHEET.md`
- One per technology
- Quick reference only

### Guide Files (Root Level)
- Format: `ACTION-TOPIC.md`
- Examples:
  - `GET-STARTED-DEVOPS.md`
  - `STUDY-PLAN-JAVA.md`
  - `RESOURCES-DEVOPS.md`

- Lowercase with hyphens
- Examples: `java-core`, `java-spring`, `docker`
- Number prefixes for ordered learning: `01-beginner`, `02-intermediate`
