## ğŸ¯ Quick Reference

### Time Complexities

| Operation | Array | Linked List | Stack | Queue | BST (avg) | Hash Table |
|-----------|-------|-------------|-------|-------|-----------|------------|
| Access | O(1) | O(n) | O(n) | O(n) | O(log n) | - |
| Search | O(n) | O(n) | O(n) | O(n) | O(log n) | O(1) |
| Insert | O(n) | O(1) | O(1) | O(1) | O(log n) | O(1) |
| Delete | O(n) | O(1) | O(1) | O(1) | O(log n) | O(1) |

### Space Complexities

- **Arrays**: O(n)
- **Linked Lists**: O(n)
- **Trees**: O(n)
- **Graphs**: O(V + E)
- **Hash Tables**: O(n)
