## ğŸ’¡ Problem-Solving Framework

### Step-by-Step Approach

1. **Understand the Problem** (5 min)
   - Read carefully
   - Ask clarifying questions
   - Identify inputs/outputs
   - Note constraints

2. **Examples & Edge Cases** (5 min)
   - Work through examples
   - Identify edge cases
   - Test assumptions

3. **Brainstorm Solutions** (5 min)
   - Brute force first
   - Think about patterns
   - Consider data structures
   - Estimate time/space complexity

4. **Code the Solution** (15-20 min)
   - Start with clean code
   - Use meaningful variable names
   - Handle edge cases
   - Add comments for complex logic

5. **Test & Debug** (5 min)
   - Test with examples
   - Check edge cases
   - Trace through code
   - Fix bugs

6. **Optimize** (5 min)
   - Analyze complexity
   - Discuss trade-offs
   - Mention alternative approaches
